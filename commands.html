Linux Command Guide

A quick reference for common Linux commands

apropos

Description: Searches manual pages for keywords. The apropos command helps you find Linux manual (man) pages related to a specific keyword. It’s useful when you don’t know the exact command name but can describe what you’re looking for.

Syntax:

apropos keyword

Example:

apropos network

alias

Description: Creates command shortcuts. The alias command allows you to create custom shortcuts for longer or complex commands, making them easier to use and saving time.

Syntax:

alias name='command'

Example:

alias la='ls -la'

uname

Description: Displays system information. The uname command provides basic system information, such as the operating system name, kernel version, and hardware details.

Syntax:

uname [options]

Example:

uname -a

df

Description: Shows disk space usage. The df command displays disk space usage of file systems, including information about available and used space.

Syntax:

df [options]

Example:

df -h

du

Description: Displays file and directory disk usage. The du command estimates and displays the disk space usage of files and directories, helping you find which items take up the most space.

Syntax:

du [options] [file/directory]

Example:

du -sh /home/user

mount

Description: Mounts a filesystem. The mount command is used to attach file systems (e.g., external drives or network shares) to the Linux directory tree, making them accessible.

Syntax:

mount [options] [device] [directory]

Example:

mount /dev/sda1 /mnt

ln

Description: Creates links between files (hard or symbolic). The ln command creates links between files, either as hard links (direct pointers) or symbolic links (shortcuts).

Syntax:

ln [options] target link_name

Example:

ln -s /path/to/file /path/to/symlink

man

Description: Displays command manual pages. The man command displays the manual pages for Linux commands, providing detailed documentation, options, and examples.

Syntax:

man command

Example:

man ls

info

Description: Displays detailed command documentation.The info command provides detailed documentation about commands, often more comprehensive and structured than man pages.

Syntax:

info command

Example:

info ls
